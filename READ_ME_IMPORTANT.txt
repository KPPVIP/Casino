INSTALLATION
1. https://forum.cfx.re/t/the-diamond-casino-heist-arcade-interiors/917315 download this map.
2. Copy to resources folder and replace the client.lua in it with the casino/client.lua in my [required] folder.
3. If you dont step 1 and 2, casino vault door always open.
4. Go to server.cfg or resources.cfg and write those.
ensure casino
ensure utk_hackdependency
ensure utk_fingerprint
ensure rm_casinoheist
5. Write the name of the casino map resource on the first line.
6. Game build must be 2060 or higher.
7. Done. The last required thing is to check the config.lua and add the items in the config.lua to the database.
